# Common Icons

To use this set of icons, import `@ode-react-ui/icons` into your code:

- `import { AddUser, AlertCircle, Check } from "@ode-react-ui/icons;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <button>
      <AddUser /> Add a new user
    </button>
  );
}
```
