# Navbars Icons

To use this set of icons, import `@ode-react-ui/icons/nav` into your code:

- `import { NeoAssistance, Search, MyApps } from "@ode-react-ui/icons/nav;"`

## Usage

Then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <nav>
      <MyApps />
    </nav>
  );
}
```
