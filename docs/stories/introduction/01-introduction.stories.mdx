[ODE React UI](https://github.com/opendigitaleducation/ode-react-ui) is the React implementation of the Open Digital Education Design System.
Our goal is to provide a consistent, accessible, and performant user experience to our users.

ODE React UI must be used with [ODE Bootstrap](?path=/docs/introduction-theming--page) for theming implementation.

## What is ODE React UI ?

Ode React UI is a monorepo with:

- Shared configuration
- 4 React Packages
- Storybook to visualize components, icons and interact with them

Many tools are already configured:

- [ReactJS](https://beta.reactjs.org/)
- [ViteJS](https://v3.vitejs.dev/guide/)
- [TypeScript](https://www.typescriptlang.org/)
- [Eslint](https://eslint.org/)
- [Prettier](https://prettier.io/)
- [Husky](https://github.com/typicode/husky)
- [Turborepo](https://turbo.build/repo)
- [Storybook](https://storybook.js.org/)

For development, please install these Visual Code Studio extensions:

- [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode)
- [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint)

Recommended Extensions:

- Auto Rename Tag : [Link](https://marketplace.visualstudio.com/items?itemName=formulahendry.auto-rename-tag)
- Better Comments : [Lien](https://marketplace.visualstudio.com/items?itemName=aaron-bond.better-comments)
- ES7+ React : [Lien](https://marketplace.visualstudio.com/items?itemName=dsznajder.es7-react-js-snippets)
