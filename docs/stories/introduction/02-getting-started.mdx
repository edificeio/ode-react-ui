# Getting Started

## Install

> ⚠️ Before starting any development, please read the [CONTRIBUTION](./CONTRIBUTING.md) Page.

```bash
pnpm install
```

> A post-installation script will run to build all packages once.

## Dev

Any available commands during your development.

## Lint

> Lint all packages from root folder

```bash
pnpm lint
```

## Prettier

> Prettier all packages from root folder

```bash
pnpm format
```

## Clean

> Clean all node_modules and lock file.

```bash
pnpm clean
```

## Build

> Build all packages from root folder

```bash
pnpm build
```

---

## Storybook

> Start Storybook on port 6006

```bash
pnpm docs
```

## [Turborepo](https://turborepo.org/)

### Cache

Turborepo will cache `build`, `lint`, `fix` and `format` results. So if no changes is detected and you run one command again, Turborepo will read from the cache and only apply the task on new changes.
