# Spacings

- Spacings are based on the 4px Grid System (it can be divided and multiplied many times and result in whole numbers).
- Spacings are used for `margin`, `padding` and `gap` CSS utilities.

**Exception**: The 4px Grid System starts at 2px to handle smaller spacings for padding and margin.

## Responsive

You can reassign spacings with breakpoints utilities: `sm`, `md`, `lg`, `xl`, and `xxl`.

Format: ` *-{direction}-{breakpoint}-{value}`
