# Getting Started

To use this set of icons, import `@ode-react-ui/icons` into your code:

- named import `import { AddUser, AlertCircle, Check } from "@ode-react-ui/icons;"`

Best practice: Named import is the best approach to import icons.

## Usage

If you use named import, then you can use any icon as follows:

```jsx
function MyComponent() {
  return (
    <button>
      <AddUser /> Add a new user
    </button>
  );
}
```
