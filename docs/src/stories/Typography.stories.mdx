## Common Text

Body

```
<p>Body</p>
```

**Body Bold**

```
<p><strong>Body Bold</strong></p>
```

_Body Italic_

```
<p><em>Body Italic</em></p>
```

**_Body Bold Italic_**

```
<p><strong><em>Body Bold Italic</em></strong></p>
```

Small

```
<p className="small">Small</p>
```

**Small Bold**

```
<p className="small"><strong>Small Bold</strong></p>
```

_Small Italic_

```
<p className="small"><em>Small Italic</em></p>
```

**_Small Bold Italic_**

```
<p className="small"><strong><em>Small Bold Italic</em></strong></p>
```

Caption

```
<p className="caption">Caption</p>
```

**Caption Bold**

```
<p className="caption"><strong>Caption Bold</strong></p>
```

_Caption Italic_

```
<p className="caption"><em>Caption Italic</em></p>
```

**_Caption Bold Italic_**

```
<p className="caption"><strong><em>Caption Bold Italic</em></strong></p>
```
