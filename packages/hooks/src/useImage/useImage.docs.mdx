# useTitle

## Usage

- Give the ability to detect and use the current Document title.
- Currently used in Responsive with our Header Component.

## Function

```jsx
function useTitle() {
  const documentDefined = typeof document !== "undefined";

  const [title, setTitle] = useState < string > (() => document.title);

  useEffect(() => {
    if (!documentDefined) return;

    setTitle(document.title);
  }, []);

  return {
    title,
  };
}
```
