# useImage

## Usage

- Use a placeholder if an error occured with the provided `src`
- Used in the `<Image/>` Component

## Example

<Canvas>
  <Story id="hooks-useimage--example" />
</Canvas>

## Function

```jsx
function useImage({ src, placeholder }: useImageProps) {
  const [imgSrc, setImgSrc] = useState < string > (src || placeholder);

  const onError = useCallback(() => {
    setImgSrc(placeholder);
  }, []);

  return { imgSrc, onError };
}
```
